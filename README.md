# Quiz App

App Demo:
https://quiz-eight-dun.vercel.app/

You can test the application by playing the quiz directly in the browser. No registration is required.

Quiz App is a simple and interactive web application designed to test users’ knowledge through a set of questions. The app provides instant feedback on answers and keeps track of the score, creating an engaging and dynamic user experience. It is built using React with a focus on clean UI and smooth interaction.

## Features
Multiple-choice quiz questions
Real-time answer feedback
Score tracking
Simple and intuitive user interface
Technologies Used
React
